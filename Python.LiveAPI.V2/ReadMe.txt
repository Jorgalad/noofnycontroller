This is the Python script to run as the Midi Remote Script (LiveAPI) for Ableton Live and the custom designed NoofnyController MIDIBOX controller.

*** Note that this is the script which is compatible with versions of Ableton Live 8 and above (tested on 8.0.4)  ***
